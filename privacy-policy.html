Lunov Privacy Policy

Last updated: October 7, 2025

1. Information We Collect

Lunov may collect User IDs, Server IDs, and message data needed for commands and moderation features. No personal data like emails or passwords is stored.

2. How We Use the Data

We use data solely to deliver bot features, improve performance, and prevent abuse.

3. Data Retention

Data is stored temporarily or securely for moderation logs. Temporary message content is deleted after processing.

4. Sharing

We do not share any user data with third parties unless required by law.

5. Security

Reasonable measures are taken to protect all stored data, but no system is completely secure.

6. User Rights

Users can request data deletion by contacting parrotgames.contact@gmail.com.

7. Updates

We may update this policy periodically. Continued use means you accept the new version.
